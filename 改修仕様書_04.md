# 改修仕様書 04

### 改修の目的

### 改修内容

#### 1. zaicoApi.jsの修正
- localStorage.getItem('zaicoApiKey')の削除
- バックエンドAPIルート経由に変更

#### 2. API呼び出しの変更
```javascript
// 修正前
const apiKey = localStorage.getItem('zaicoApiKey');

// 修正後
const response = await fetch('/api/zaico/inventories');
```

#### 3. エラーハンドリングの更新
- APIキー関連エラーの除去
- バックエンドからのエラーレスポンス処理

### 影響範囲
- `src/utils/zaicoApi.js`

### 所要時間
2-3時間

### 検証項目
