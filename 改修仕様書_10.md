# 改修仕様書 10

## フェーズ1.10 - AuthContextパスワードハッシュ化対応

### 改修の目的
認証コンテキストでパスワードハッシュ化を使用するように変更する

### 改修内容

#### 1. AuthContext.jsxの修正
```javascript
// src/contexts/AuthContext.jsx
import { hashPassword, verifyPassword } from '../utils/passwordHash';

const AuthContext = createContext();

export const useAuth = () => {
  // login関数の修正
  // register関数の修正
  // パスワード検証ロジックの変更
}
```

#### 2. ログイン処理の変更
- 平文パスワード比較を廃止
- ハッシュ化パスワードでの検証に変更

#### 3. 登録処理の変更
- 新規ユーザー登録時のパスワードハッシュ化
- ハッシュ化されたパスワードの保存

### 影響範囲
- `src/contexts/AuthContext.jsx`

### 所要時間
3-4時間

### 検証項目
- 新規登録でパスワードがハッシュ化されること
- ログインで正しい認証ができること
- 平文パスワードでのログインができないこと
