# 改修仕様書 13

## フェーズ1.13 - Loginコンポーネント更新

### 改修の目的
ログインコンポーネントでパスワードハッシュ化対応を完了する

### 改修内容

#### 1. Login.jsxの更新
- AuthContextの新しいlogin関数を使用
- 移行関連のメッセージ表示

#### 2. ユーザーフィードバック
```jsx
// Login.jsx
const handleLogin = async (e) => {
  try {
    const result = await login(email, password);
    
    if (result.migrated) {
      showMessage('セキュリティ向上のため、パスワードを更新しました');
    }
    
    navigate('/dashboard');
  } catch (error) {
    setError(error.message);
  }
}
```

#### 3. エラー処理の改善
- 認証失敗の明確なメッセージ
- 移行関連エラーの専用処理

### 影響範囲
- `src/components/Login.jsx`

### 所要時間
1-2時間

### 検証項目
- ログイン機能が正常動作すること
- 移行メッセージが適切に表示されること
- エラーメッセージが分かりやすいこと
