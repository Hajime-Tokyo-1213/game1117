# 改修仕様書 08

## フェーズ1.8 - localStorage APIキーデータ削除

### 改修の目的
既存のlocalStorageからAPIキー情報を完全に削除する

### 改修内容

#### 1. データクリーンアップユーティリティの作成
```javascript
// src/utils/dataCleanup.js
export const cleanupApiKeyData = () => {
  localStorage.removeItem('zaicoApiKey');
  localStorage.removeItem('apiKey');
  localStorage.removeItem('zaico_api_key');
  // その他のAPIキー関連データ
}
```

#### 2. 起動時の自動クリーンアップ
- アプリケーション起動時にAPIキーデータを削除
- 一回限りの実行フラグ管理

#### 3. マイグレーション機能
```javascript
// src/utils/migration.js
export const runMigrations = () => {
  const migrationVersion = localStorage.getItem('migrationVersion') || '0';
  
  if (migrationVersion < '1.0') {
    cleanupApiKeyData();
    localStorage.setItem('migrationVersion', '1.0');
  }
}
```

### 影響範囲
- 新規: `src/utils/dataCleanup.js`
- 新規: `src/utils/migration.js`
- `src/main.jsx`（起動時の実行）

### 所要時間
1-2時間

### 検証項目
- APIキー関連データが完全に削除されること
- アプリケーション機能に影響がないこと
- マイグレーションが一回のみ実行されること
