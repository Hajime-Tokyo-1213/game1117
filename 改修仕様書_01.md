# 改修仕様書 01

## フェーズ1.1 - 環境変数による基盤設定

### 改修の目的
セキュリティ強化の基盤として環境変数の管理を整備する

### 改修内容

#### 1. 環境設定ファイルの作成
- `.env.example`ファイルの作成
- 本番環境用の環境変数テンプレート

#### 2. 環境変数設定項目
```env
# API Keys
ZAICO_API_KEY=your_zaico_api_key_here

# Database
DATABASE_URL=your_database_url_here

# JWT
JWT_SECRET=your_jwt_secret_here
JWT_EXPIRES_IN=7d

# App
NODE_ENV=production
PORT=3000
```

#### 3. Viteの環境変数設定
- `vite.config.js`の更新
- 環境変数の適切な露出設定

### 影響範囲
- プロジェクトルート（.env関連ファイル）
- `vite.config.js`
- `.gitignore`

### 所要時間
2-3時間

### 検証項目
- 環境変数が正しく読み込まれること
- 機密情報がクライアントサイドに露出しないこと
